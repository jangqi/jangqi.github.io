*p = 1； /*指针变量的类型决定访问内存地址的延续长度*/

p+2;  /*从当前地址向后移动2个int单元（8字节）*/
p-2;
p++;

解地址

空指针：值为NULL的指针变量叫空指针，如果进行解引用就会产生段错误。

NULL也是一种错误标志，如果一个函数返回值是指针类型，当函数执行出错返回值就是NULL。

当函数的参数是指针，别人传给你的就有可能是空指针。

从函数获取的返回值也可能是空指针。

如何避免空指针带来的段错误：使用来历不明的指针前做判断。if(NULL == p)

注意：NULL在绝大多数系统中是0，在个别系统中是1。

野指针：指向不确定的内存空间。

如何避免产生野指针：

1.定义指针变量时一定初始化。

2.函数不返回栈内存的地址。

3.指针指向的内存释放后，指针变量要及时置空。

数组与指针

数组

从内存角度，数组变量就是一次分配多个变量，而且这些变量的存储地址是依次相连的。

数组表示法和指针表示法

数组下标方式和指针方式均可以访问数组元素，两者的实质其实是一样的。在编译器内部都是用指针方式来访问数组元素的，数组下标方式只是编译器提供给编程者一种壳（语法糖）而已。用指针方式来访问数组才是本质做法。

/** 数组表示法*/
/** 作为函数参数的数组名总是编译器转化为指针*/
void F_Array(int arr[], int n)	/*int array[]等同于指针int *arr形式*/
{
	arr[1] = 1;	  /* arr[1]等同于*(arr+1)*/
}

/** 指针表示法*/
void F_Array_P(int* arr, int n)     /*指向int类型的指针和偏移量n做形式参数*/
{
	*(arr+1) = 1  /*从内存arr位置，移动1个int单元，对下一个int单元赋值*/
}

结构体作为函数的参数时，使用结构体指针效率更高。

指针访问结构成员变量的两种方式：指针->运算符和(*指针).运算符等效。
