c关键字

const

定义只读变量， const 修饰的变量的值不允许改变，即不允许给它重新赋值。

数组的长度不能是变量，虽然 const 定义的是只读变量，相当于定义一个常量。但是只读变量也是变量，所以 const 定义的变量仍然不能作为数组的长度。

1
2

const  int  a = 10;
int  const  a = 10;  /** 等同*/

define

define定义的是常量，define是预编译指令，define定义的宏是在预处理阶段进行文本替换，编译后就不存在了，因为它不是变量，系统只会给变量分配内存。

const定义的只读变量本质上仍然是一个变量，具有变量的基本属性，有类型、占用内存。

static

修饰局部变量（静态局部变量），改变了局部变量的存储位置，即从原来的栈中存放改为静态存储区存放，所以静态局部变量的值在函数调用结束之后不会消失，而仍然保留其原值。

修饰全局变量（静态全局变量），限制其作用域只在定义该变量的源文件内有效，其他源文件中不能使用。

修饰一个函数，限制其作用域只在定义该函数的源文件内有效，其他源文件中不能使用。

extern

在一个源文件中想引用另外一个源文件中已经定义的外部变量，需要用 extern 加以声明，从声明处起，合法地使用该外部变量。

对变量而言，如果想一个本源文件中使用另一个源文件已经定义的外部变量，就需要在使用前用extern声明该变量（或者在头文件中用extern声明）。

对函数而言，如果想一个本源文件中使用另一个源文件已经定义的函数，就需要在使用前用声明该函数，声明函数加不加extern都没关系，所以在头文件中函数可以不用加extern。

void 指针用法，

void 指针是无类型指针，不允许进算术运算，无类型的指针进行算术运算时，不知道该操作几个字节（数据长度）。

可以用任意类型的指针对 void 指针赋值。

但如果要将 void 指针赋给其他类型的指针，则需要强制类型转换。需要显式说明该指针指向的内存中是存放什么类型的数据。

1

 (int *)malloc(1024);  /** 强制转换 malloc() 返回的 void* 指针指向的内存中存放的是一个个的 int 型数据*/

典型的如内存操作函数 memcpy 和 memset，任何类型的指针都可以传入 memcpy 和 memset 中，这也真实地体现了内存操作函数的意义，因为它操作的对象仅仅是一片内存，而不论这片内存是什么类型。

1
2

void * memcpy(void *dest, const void *src, size_t len);
void * memset ( void * buffer, int c, size_t num );
